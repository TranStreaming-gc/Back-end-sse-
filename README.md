# Back-end-sse-

## 현재 과업
1. **사용자 정보 업데이트**
   - Spring Data JPA를 활용하여 사용자 정보를 쉽게 업데이트할 수 있는 기능 구현.
   - RESTful API를 통해 사용자 정보 관리.

2. **메세지 상태 개선**
   - 지속적으로 발생하는 알람을 단일 알람으로 변환하는 로직 구현.
   - 중복 알람 방지 및 사용자 편의성 증대.

3. **알람 내용 커스텀 기능**
   - 사용자가 알람 내용을 직접 설정하고 수정할 수 있는 기능 추가.
   - 다양한 알람 유형에 대한 사용자 맞춤화 지원.
